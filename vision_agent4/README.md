
## Image Effects Project

### 개발목적
본 프로그램은 이미지에 특수효과를 넣어 사용자가 다양한 질감의 이미지를 볼 수 있게 한다.

### 기능소개
- 엠보싱 표현<br>
- 스케치 질감(컬러,흑백) 표현<br>
- 유화 질감 표현<br>
- 카툰 질감 표현<br>

### 개발환경
```bash
python -V #3.10.10
pip install PyQt6 #6.7.1 
pip install opencv-python #4.10 
pip install opencv-contrib-python # 4.10.0.84
```

### 실행 방법

#### 로컬
```bash
python src/main.py
```
#### 실행파일
```
dist/SpecialEffectApp.exe
```
